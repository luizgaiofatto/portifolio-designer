# Emily Designer

Website de portfólio moderno e responsivo desenvolvido para apresentar o trabalho de uma designer de forma elegante, profissional e intuitiva.

## Sobre o projeto

Site criado com o objetivo de destacar projetos, habilidades e informações de contato de uma designer, proporcionando uma navegação simples e uma experiência visual agradável.

O layout foi pensado para ser limpo, organizado e totalmente adaptável a diferentes dispositivos (desktop, tablet e mobile).

## Funcionalidades

- Apresentação da designer  
- Exibição de projetos (portfólio)  
- Design responsivo  
- Links para redes sociais  
- Contato direto via email e WhatsApp  
- Navegação entre seções  

## Tecnologias

- HTML5  
- CSS3  
- JavaScript
